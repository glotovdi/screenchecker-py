# screenchecker-py

Скрипт воспроизведения аудиофайла, в случае нахождения заданного текста на экране

1. Установить зависимости `pip3 install -r requirements.txt`
2. Установить https://github.com/UB-Mannheim/tesseract/wiki
3. Положить в корень Tesseract RU языковой пакет https://github.com/tesseract-ocr/tessdata/raw/main/rus.traineddata
4. Поместить файл для воспроизведения C:/sound.wav
5. Запустить bat файл, выбрать монитор, интервал и текст для поиска
