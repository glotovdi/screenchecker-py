# Screen Checker

Приложение для мониторинга экрана с распознаванием текста (OCR). При обнаружении заданного текста на экране воспроизводит звуковое оповещение и сохраняет скриншот.

## Структура проекта

```
screenchecker-py/
├── data/                      # Данные и конфигурация
│   ├── config.json           # Настройки приложения
│   ├── screenshots/          # Сохранённые скриншоты
│   └── logs/                 # Логи работы
├── assets/
│   └── sounds/               # Звуковые файлы
├── src/                      # Исходный код
│   ├── main.py              # Точка входа
│   ├── console_mode.py      # Консольный режим
│   ├── core/                # Ядро приложения
│   │   ├── capture.py       # Захват экрана
│   │   ├── check_config.py  # Проверка конфигурации
│   │   ├── logger_setup.py # Логирование
│   │   ├── ocr_processor.py # OCR обработка
│   │   └── region_selector.py # Выбор области экрана
│   └── gui/                 # Графический интерфейс
│       └── app.py           # Tkinter приложение
├── requirements.txt          # Python зависимости
├── install.bat             # Установка зависимостей
├── check_config.bat       # Проверка конфигурации
├── run_gui.bat           # Запуск GUI версии
└── run_console.bat       # Запуск консольной версии
```

## Установка

### 1. Установка Tesseract OCR

Скачайте и установите Tesseract с официального репозитория:
- https://github.com/UB-Mannheim/tesseract/wiki

### 2. Установка русского языкового пакета

Скачайте файл `rus.traineddata`:
- **Прямая ссылка:** https://github.com/tesseract-ocr/tessdata/raw/main/rus.traineddata

После скачивания поместите файл в папку `tessdata`:
```
C:\Program Files\Tesseract-OCR\tessdata\rus.traineddata
```

### 3. Установка Python зависимостей

Запустите командный файл:
```cmd
install.bat
```

Или вручную:
```cmd
pip install -r requirements.txt
```

### 4. Проверка конфигурации

Запустите:
```cmd
check_config.bat
```

## Запуск

### GUI режим (рекомендуется)
```cmd
run_gui.bat
```

### Консольный режим
```cmd
run_console.bat
```

## Использование

### Выбор области мониторинга
1. Выберите монитор в выпадающем списке
2. Поставьте галочку **"Область"**
3. Нажмите **"Выбрать область"**
4. Нарисуйте прямоугольник на нужном участке экрана
5. Можно выбрать область на любом мониторе

### Настройка звука
- **Выбрать** — выбор звукового файла (формат WAV)
- **Тест** — воспроизвести звук с текущими настройками
- **Повторы** — количество повторов звука (1-10)

### Запуск мониторинга
1. Выберите монитор (при необходимости — область)
2. Установите интервал проверки (секунды)
3. Введите текст для поиска
4. Выберите звуковой файл и настройте повторы
5. Нажмите **Старт**

### Когда текст найден:
- Воспроизводится звуковое оповещение
- Сохраняется скриншот экрана в `data/screenshots/`
- Записывается событие в лог

### Остановка:
- **Стоп** — остановка мониторинга
- **Сброс работы** — полный сброс (остановка + обнуление счётчиков)
- **Сброс логов** — очистка окна логов
- **Удалить скриншоты** — удаление всех сохранённых скриншотов

## Конфигурация

Файл `data/config.json`:

```json
{
    "tesseract_path": "C:\\Program Files\\Tesseract-OCR\\tesseract.exe",
    "screenshots_dir": "data\\screenshots",
    "log_dir": "data\\logs",
    "default_interval": 5,
    "ocr_config": "--oem 3 --psm 7",
    "image_scale": 2.0,
    "sound_file": ""
}
```

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `tesseract_path` | Путь к Tesseract | - |
| `sound_file` | Путь к звуку (задаётся через GUI) | "" |
| `screenshots_dir` | Папка скриншотов | `data\screenshots` |
| `log_dir` | Папка логов | `data\logs` |
| `default_interval` | Интервал проверки (сек) | `5` |
| `ocr_config` | Параметры Tesseract | `--oem 3 --psm 7` |
| `image_scale` | Масштаб для OCR | `2.0` |

## Логи

Логи хранятся в `data/logs/` с дневной ротацией:
- Формат: `scan_YYYYMMDD.log`
- Максимальный размер: 5 МБ
- Резервных копий: 5

## Требования

- Windows 10/11
- Python 3.8+
- Tesseract OCR 5.x
- Русский языковой пакет для Tesseract

## Возможные проблемы

### Tesseract не найден
Проверьте путь в `config.json`.

### OCR не распознаёт текст
- Увеличьте `image_scale` в конфиге
- Используйте более крупный шрифт
- Проверьте наличие русского языкового пакета

### Звук не воспроизводится
- Файл должен быть в формате **WAV**
- Проверьте наличие файла
- Для громкости используйте системные настройки Windows
