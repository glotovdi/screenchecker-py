# Screen Checker

Приложение для мониторинга экрана с распознаванием текста (OCR). При обнаружении заданного текста на экране воспроизводит звуковое оповещение и сохраняет скриншот.

## Структура проекта

```
screenchecker-py/
├── data/                      # Данные и конфигурация
│   ├── config.json           # Настройки приложения
│   ├── screenshots/          # Сохранённые скриншоты
│   └── logs/                 # Логи работы
├── assets/
│   └── sounds/               # Звуковые файлы
│       └── sound.wav         # Звук оповещения
├── src/                      # Исходный код
│   ├── main.py              # Точка входа
│   ├── console_mode.py      # Консольный режим
│   ├── core/                # Ядро приложения
│   │   ├── capture.py       # Захват экрана
│   │   ├── logger_setup.py  # Логирование
│   │   └── ocr_processor.py # OCR обработка
│   └── gui/                 # Графический интерфейс
│       └── app.py           # Tkinter приложение
├── requirements.txt          # Python зависимости
├── install.bat               # Установка зависимостей
├── check_config.bat         # Проверка конфигурации
├── run_gui.bat              # Запуск GUI версии
└── run_console.bat          # Запуск консольной версии
```

## Установка

### 1. Установка Tesseract OCR

Скачайте и установите Tesseract с официального репозитория:
- https://github.com/UB-Mannheim/tesseract/wiki

При установке выберите русский языковой пакет (Russian).

### 2. Установка Python зависимостей

Запустите командный файл:
```cmd
install.bat
```

Или вручную:
```cmd
pip install -r requirements.txt
```

### 3. Настройка конфигурации

Откройте файл `data/config.json` и укажите пути:

```json
{
    "tesseract_path": "C:\\Program Files\\Tesseract-OCR\\tesseract.exe",
    "sound_file": "assets\\sounds\\sound.wav",
    "screenshots_dir": "data\\screenshots",
    "log_dir": "data\\logs",
    "default_interval": 5,
    "ocr_config": "--oem 3 --psm 6",
    "image_scale": 1.5,
    "min_confidence": 60
}
```

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `tesseract_path` | Путь к исполняемому файлу Tesseract | - |
| `sound_file` | Путь к звуковому файлу оповещения | `assets\sounds\sound.wav` |
| `screenshots_dir` | Папка для сохранения скриншотов | `data\screenshots` |
| `log_dir` | Папка для логов | `data\logs` |
| `default_interval` | Интервал проверки (секунды) | `5` |
| `ocr_config` | Параметры Tesseract | `--oem 3 --psm 6` |
| `image_scale` | Масштаб изображения для OCR | `1.5` |

### 4. Подготовка звукового файла

Поместите звуковой файл в формате WAV по пути:
```
assets\sounds\sound.wav
```

### 5. Проверка конфигурации

Запустите:
```cmd
check_config.bat
```

Скрипт проверит:
- Наличие Tesseract
- Наличие звукового файла
- Создание необходимых папок

## Запуск

### GUI режим (рекомендуется)

```cmd
run_gui.bat
```

Откроется графический интерфейс с возможностью:
- Выбора монитора
- Установки интервала проверки
- Ввода текста для поиска
- Просмотра статистики и логов

### Консольный режим

```cmd
run_console.bat
```

Интерактивный режим через командную строку.

## Использование

1. **Выберите монитор** — укажите номер монитора для мониторинга
2. **Установите интервал** — частота проверок в секундах (минимум 1)
3. **Введите текст** — текст для поиска на экране
4. **Нажмите Старт** — начнётся мониторинг

### Когда текст найден:
- Воспроизводится звуковое оповещение
- Сохраняется скриншот экрана в `data/screenshots/`
- Записывается событие в лог

### Остановка:
- Нажмите **Стоп** в GUI
- Или **Ctrl+C** в консольном режиме

## Логи

Логи хранятся в папке `data/logs/` с дневной ротацией:
- Формат: `scan_YYYYMMDD.log`
- Максимальный размер одного файла: 5 МБ
- Количество резервных копий: 5

## Требования

- Windows 10/11
- Python 3.8+
- Tesseract OCR 5.x
- Русский языковой пакет для Tesseract

## Возможные проблемы

### Tesseract не найден
Убедитесь, что путь в `config.json` указывает на правильный файл `tesseract.exe`.

### OCR не распознаёт текст
- Увеличьте `image_scale` в конфиге
- Используйте более крупный шрифт на экране
- Проверьте наличие русского языкового пакета

### Звук не воспроизводится
- Проверьте формат файла (должен быть WAV)
- Убедитесь, что файл существует по указанному пути
